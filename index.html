โครงการศูนย์รวบรวมและคัดแยกวัสดุรีไซเคิล (สมุทรสาคร)

เป้าหมาย: Hub ระดับจังหวัด
หน้านี้คือสรุปสำหรับตัดสินใจลงทุน: ปริมาณวัสดุรีไซเคิลทั้งจังหวัด (ซัพพลาย), ลูกค้าโรงงานปลายทาง, ศักยภาพการขนส่ง, และความเสี่ยงเชิงพื้นที่/ชุมชน โดยโฟกัสจุดปฏิบัติการ “ลานกระทิงแดง” ว่าจะเป็นจุดรวมกลางของห่วงโซ่รีไซเคิลจังหวัดได้หรือไม่
กำลังผลิตที่ตั้งเป้า
≈ 30,000 ตัน / ปี
โมเดลธุรกิจ
รวบรวม → คัดแยก → อัดก้อน → ส่งโรงงาน
ลูกค้าหลัก
กระดาษ / เม็ดพลาสติก / โรงงานสุรา
สถานะ
กำลังวางโมเดลปฏิบัติการ
♻️
1. ปริมาณวัสดุรีไซเคิลในจังหวัด (Supply) ซัพพลายหนาแน่น
วัสดุหลัก: PET / HDPE / กระดาษลูกฟูก / กระป๋องอะลูมิเนียม / ขวดแก้วใช้ซ้ำ
โซนแรงงาน & อุตสาหกรรมหนาแน่น: นาดี / ท่าทราย / พันท้ายนรสิงห์
ศักยภาพรวม (ประเมิน)
~30,000 ตันต่อปี
จุดสังเกต
  • จังหวัดมีโรงงานและคลังที่ปล่อยวัสดุรีไซเคิลต่อเนื่องทั้งปี
  • กระดาษกล่อง / พลาสติกแข็งเป็นสัดส่วนใหญ่ของปริมาณ
  • ขวดแก้วสะอาด = ของพรีเมียม ส่งกลับโรงงานสุราได้ (ขายเป็น “ใบ” ไม่ใช่กิโล)
จุดโรงงานและความหนาแน่นพื้นที่ผลิตของเสีย → ดูแผนที่
ตัวเลขตามประเภทวัสดุ → ประมาณการสินค้า
🏭
2. ลูกค้าโรงงานปลายทาง & คู่แข่งรอบไซต์ ต้องแย่งวัตถุดิบ
ลูกค้าหลัก: โรงงานกระดาษ / โรงงานเม็ดพลาสติก / โรงงานสุรา (รับขวดแก้วหมุนเวียน)
ภายในรัศมี 5-15 กม. มีทั้ง “ลูกค้า” และ “ลานของเก่า/รีไซเคิล” ที่เป็นคู่แข่งโดยตรง
จุดแข็งของเรา
  • ตั้งเป็นศูนย์กลางเดียว → รับ, คัดแยก, อัดก้อน, ส่งทีเดียว
  • โรงงานปลายทางไม่ต้องรับรถเล็กหลายรอบ ลดภาระรับของกระจัดกระจาย
ความท้าทาย
  • ผู้เล่นรีไซเคิลเดิมอยู่ใกล้จุดตั้ง → แข่ง “ราคา” + “ความเร็วในการยกของ”
  • ต้องสร้างภาพว่าเป็น “ศูนย์อุตสาหกรรม” ไม่ใช่ “ลานของเก่า” ธรรมดา
รายชื่อคู่แข่งรอบไซต์ + วัดระยะทางจริง → คู่แข่ง / ลูกค้า
🚚
3. โลจิสติกส์ & การขนส่ง เข้าถึงง่าย
รถบรรทุกหนักเข้าได้ ใกล้ถนนเศรษฐกิจ อยู่กลางโซนอุตสาหกรรม
ระยะส่งลูกค้าหลักส่วนใหญ่ < 20 กม.
ต้นทุนขนส่งต่อกิโลต่ำ
ความสามารถรถ
  • รองรับรถบรรทุก ~30 ตัน / เที่ยว (อัดก้อนเต็มพาเลท)
  • วิ่งน้อยรอบ → ลดต้นทุนต่อกิโล → เพิ่มกำไรต่อตัน
โอกาสขยาย
  • ไซต์นี้ตั้งเป็น Hub กลาง: เก็บจากชุมชน/โรงงานรอบนอก → รวม/อัด → ยิงตรงโรงงานปลายทาง
จุดโหลด/จอดรถ และเส้นทางส่งจริง → การขนส่ง
ความสามารถเครื่องอัด (แนวนอน/แนวตั้ง) → กำลังการผลิต
⚠️
4. ความเสี่ยงพื้นที่ / ชุมชน / น้ำท่วม ต้องควบคุมภาพลักษณ์
ถ้าชุมชนมองว่า “สกปรก / เสียงดัง / รถใหญ่รบกวน” โปรเจกต์อาจโดนต่อต้านได้
เขตลุ่มปากแม่น้ำท่าจีน → เสี่ยงน้ำฝนขัง / น้ำทะเลหนุน
ประเด็นสิ่งแวดล้อม / สังคม
  • ชุมชนใกล้ไซต์ → ต้องคุมเสียงรถกลางคืน
  • ฝุ่นกระดาษ / กลิ่นพลาสติกอุ่นตอนอัดก้อน ต้องจัดการ
  • ภาพภายนอกต้องเหมือน “ศูนย์คัดแยกมาตรฐานอุตสาหกรรม” ไม่ใช่ “กองขยะ”
น้ำท่วม / ภูมิประเทศ
  • พื้นที่ค่อนข้างต่ำ → ฝนหนักมีสิทธิน้ำขัง
  • ต้องวางกองวัตถุดิบบนพื้นยกสูง ไม่ให้ของเสียมูลค่า
สถานะความเสี่ยงแบบไฟจราจร 🟢🟡🔴 และรายละเอียดชุมชน → ความเสี่ยง

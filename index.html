ศูนย์กลางรวบรวมวัสดุรีไซเคิล (“ลานกระทิงแดง”)

โมเดลรายได้รีไซเคิลอุตสาหกรรม

เป้าหมาย: ตั้งฮับกลางในสมุทรสาคร รับวัสดุรีไซเคิลจากแรงงาน/โรงงานรอบจังหวัด → คัดแยก → อัดก้อนมาตรฐาน → ส่งให้โรงงานปลายทางโดยตรง จุดขายหลัก:

  • ขวดแก้วดี = ส่งกลับโรงงานสุราแบบ “ใช้ซ้ำเลย” ขายเป็นใบ (ไม่ต้องบด)
  • PET / HDPE / กระดาษลูกฟูก / กระป๋องอลูมิเนียม = อัดก้อนแนวนอน ขนาดมาตรฐานโรงงาน
  • รวมขึ้นรถใหญ่เที่ยวละ ~30 ตัน → ลดต้นทุนขนส่งต่อกิโล

วัสดุเข้าระบบ (Supply) ของเข้าเยอะ

PET / HDPE / กระดาษลูกฟูก / กระป๋องอลู / ขวดแก้ว

  • จังหวัดอุตสาหกรรม → มีของเหลือทิ้งทุกวัน
  • แรงงานต่างด้าวหนาแน่น = สายเก็บ/สายขายของเก่าสร้างซัพพลายเอง
  • วัตถุดิบเกิดในจังหวัด ไม่ต้องรอป้อนจากกรุงเทพฯ
🏭

ลูกค้าปลายน้ำ (Demand) ตลาดพร้อมซื้อ

โรงงานกระดาษ / หลอมเม็ดพลาสติก / โรงงานสุรา

  • ขวดแก้วดี = ส่งกลับโรงงานสุราแบบใช้ซ้ำ (ขายเป็นใบ)
  • PET / กระดาษ / อะลูมิเนียม = อัดก้อนแล้วขาย
  • ลูกค้าปลายน้ำอยู่รัศมีใกล้ → ระบายสต็อกเร็ว ไม่จมสต็อก
🚚

โลจิสติกส์ / ทางเข้าออก ต้องจัดดี

โซนอุตสาหกรรม ไม่ใช่กลางชุมชน

  • รถใหญ่เข้าได้ → ขนทีเดียว ~30 ตัน/เที่ยว
  • ลดต้นทุนขนส่งต่อกิโล = คีย์ของกำไร
  • ต้องดีไซน์จุดกลับรถ/จอดรอ ไม่ให้ไปกวนบ้านคน
🚨

ความเสี่ยงพื้นที่ (Risk) ต้องคุมภาพลักษณ์

น้ำท่วม / กลิ่น / รถบรรทุก / คู่แข่งรอบข้าง

  • ห้ามดูเหมือน “ลานของเก่าเถื่อน”
  • ต้องโชว์ว่าคือ “ศูนย์คัดแยกอุตสาหกรรมสะอาด”
  • ยกพื้น, มีร่องน้ำ, มีระบบจัดการคราบ/น้ำเสีย ตั้งแต่วันแรก
📦
กำลังการผลิต (Capacity / Operation Model)
  • ใช้ “เครื่องอัดแนวนอน” (horizontal baler) → ป้อนต่อเนื่อง, ก้อนสวย ขายง่าย โรงงานชอบ
  • สมมติปริมาณรวมทั้งปีตอนเริ่ม: 1 เครื่องหลัก (~25 kW) + ทีมปฏิบัติการ 2 คน/กะ → ยังไม่ตันทันที = ยังมี room ขยาย
  • อัดก้อนขึ้นรถใหญ่ ~30 ตัน/เที่ยว → เราส่งออกแบบอุตสาหกรรม ไม่ใช่ขายปลีกทีละกิโล
  • Positioning: คุณคือ “Hub คัดแยกอุตสาหกรรม” ไม่ใช่ “ลานของเก่า”
📝
Check list ก่อนลงเงินจริง
  • รัศมีคู่แข่ง: ลานรับซื้อยิบย่อยเยอะมาก → กลยุทธ์เราคือ บริการรับถึงจุด + จ่ายเงินไว
  • เสียง/กลิ่น/น้ำเสีย: ต้องคุมภาพลักษณ์สะอาด ไม่ให้ชุมชนมองว่าเป็นกองขยะเถื่อน
  • ความเสี่ยงน้ำท่วม: ต้องยกพื้น, ทำรางระบายน้ำ, กันของลอยตอนฝนหนัก
  • ใบอนุญาต/กฎหมาย: branding ต้องเป็น “ศูนย์จัดการรีไซเคิล” ไม่ใช่ “กองของเก่า”
